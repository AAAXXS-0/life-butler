# 五层记忆系统 Skill

> 统一规范 agent 的记忆写入行为。核心原则:**原封不动,不加工不摘要**。

## 文件结构

| 层级 | 文件路径 |
|------|---------|
| 瞬时 | `memory/transient.md` |
| 日记 | `memory/days/YYYY-MM-DD.md` |
| 周记 | `memory/weeks/YYYY-Www.md` |
| 月记 | `memory/months/YYYY-MM.md` |
| 年记 | `memory/years/YYYY.md` |

---

## 触发规则

### 每次操作后 → 写 transient

每次操作完成后，立即写入 `memory/transient.md`，格式：

```
什么操作 / 操作何处 / 操作前后状态
```

**格式要求：**
- 用 `/` 分隔三个部分，中间不加多余标点
- 操作：动词短语（调用工具、发送消息、读写文件等）
- 操作何处：工具名、文件路径、API 端点
- 操作前后状态：`操作前 → 操作后`，无变化写"无"

**示例：**
```markdown
调用天气查询 / wttr.in API / 操作前：无 / 操作后：获取到上海多云转晴22-28度
写入行程文档 / trip-plan.md / 操作前：空文件 / 操作后：已写入北京3日游行程
```

### 每轮对话结束后 → 写日级记忆 + 清空 transient

**顺序不可颠倒：**

1. **先写 transient**：操作完成后立即写入 `memory/transient.md`
2. **对话结束时写日级记忆**：将本轮对话原文 + transient 所有操作合并写入 `memory/days/YYYY-MM-DD.md`
3. **最后清空**：清空 `memory/transient.md`

## 日级记忆格式

**核心原则:原封不动,一字不改。**

```
[HH:MM:SS] <发送者>: 原始消息
[HH:MM:SS] <发送者>: 原始消息
[操作] 操作描述 / 操作位置 / 操作前 / 操作后
[HH:MM:SS] <发送者>: 原始回复
[操作] 操作描述 / 操作位置 / 操作前 / 操作后
---
```

**关键点:**
- 消息原文一字不改(语气、标点、表情全部保留)
- 回复原文一字不改(用户看到什么就记什么)
- 操作记录只从 transient 复制,不在日级记忆里临时补写
- 对话段之间用 `---` 分隔

**示例:**
```markdown
23:41:12 <用户668149>: 帮我查一下明天天气
23:41:13 <Trip Agent>: 正在查询明天上海天气...
[操作] 调用 weather skill 查询上海天气 / wttr.in API / 操作前:无 / 操作后:获取到天气数据
23:41:15 <Trip Agent>: 明天上海多云转晴,22-28度
[操作] 将天气数据返回给用户 / 飞书消息发送 / 操作前:等待 / 操作后:已发送
---
```

## 周期汇总(周/月/年)

Agent 每次被调用时,先检查当前日期,决定是否触发周期汇总。

### 周记汇总

**触发时机:** 每周一 00:00(检查当周是否已汇总)

**逻辑:**
1. 获取本周一 ~ 今日所有 `memory/days/` 文件
2. 按主题归纳:主要事项 / 重要决策 / 待跟进 / 主题记忆
3. 写入 `memory/weeks/YYYY-Www.md`
4. 已汇总则跳过(文件存在则不重复写入)

### 月记汇总

**触发时机:** 每月最后一天 或 每月 1 日(检查当月是否已汇总)

**逻辑:**
1. 获取当月所有 `memory/weeks/` 文件
2. 归纳整月概要 / 重要事项 / 趋势观察 / 跨主题记忆
3. 写入 `memory/months/YYYY-MM.md`
4. 已汇总则跳过

### 年记汇总

**触发时机:** 每年 12 月 31 日 或 每年 1 月 1 日(检查当年是否已汇总)

**逻辑:**
1. 获取当年所有 `memory/months/` 文件
2. 归纳年度概要 / 重大事项 / 成长与变化 / 长期记忆
3. 写入 `memory/years/YYYY.md`
4. 已汇总则跳过

### 年末永久记忆同步提示

年记写完后,提醒用户:

> 年记已生成,请检视其中的「长期记忆」section,将值得永久保留的内容手动同步到 `MEMORY.md`,此外,祝您新年快乐。

永久记忆靠手动 + 被动写入,不做自动化。

### 汇总原则

- **主题式归纳**,不是流水账
- **保留来源标注**:周记引用日记,月记引用周记,年记引用月记
- **文件存在即跳过**,不重复写入
- **格式参考** `TEMPLATES/week-template.md` / `month-template.md` / `year-template.md`

---

## 注意事项

- **不加工不摘要**:对话原文直接复制,不改写
- **操作记录实时写**:工具返回后立即写 transient,不要等对话结束再补
- **文件路径固定**:不随对话切换而切换,每次操作都在同一个 transient 文件追加
- **清空时机严格**:只有每轮对话结束才清空 transient,不要中途清
- **汇总不遗漏**:每次被调用时顺手检查日期,漏掉一次就等下一次触发
